В этой папке находятся примеры входных файлов и эталонные выходные данны для них.
[N]startend.txt - координаты начала и конца пути для теста N
[N]objects.txt - координаты препятствий для теста N
[N]out.txt - эталон выходных данных для теста N

Для запуска теста нужно передать аргументы в следующем порядке:
WaysBuilder(.exe) [N]startend.txt [N]objects.txt

Для запуска атоматического тестирования запустите утилиту без параметров.
